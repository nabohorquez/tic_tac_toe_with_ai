Modelo

El modelo de IA utilizado en este juego de Tic-Tac-Toe está basado en Q-learning, un algoritmo de aprendizaje por refuerzo. Q-learning permite a la IA aprender a tomar decisiones óptimas a través de la experiencia, actualizando una tabla Q que representa las recompensas esperadas para cada acción en cada estado del juego.

Juego

El juego de Tic-Tac-Toe, también conocido como Tres en Raya, es un juego de mesa para dos jugadores. El objetivo es alinear tres de tus símbolos (X o O) en una fila, columna o diagonal en un tablero de 3x3. En esta implementación, el jugador humano juega contra una IA que utiliza el modelo de Q-learning para tomar sus decisiones.

Aplicación

La aplicación web está construida utilizando Flask, un microframework de Python. Flask permite crear aplicaciones web de manera sencilla y eficiente. La aplicación maneja las rutas para iniciar el juego, realizar movimientos y entrenar la IA.

Características de la Aplicación

  • Inicio del Juego: La aplicación inicializa el tablero y establece el jugador actual.
  • Movimientos: La aplicación maneja los movimientos del jugador humano y de la IA, actualizando el estado del juego.
  • Entrenamiento de la IA: La aplicación permite entrenar la IA utilizando Q-learning, mejorando su desempeño con el tiempo.

Tecnologías Utilizadas

  • Python: Lenguaje de programación utilizado para la lógica del juego y la IA.
  • Flask: Framework utilizado para construir la aplicación web.
  • HTML/CSS: Tecnologías utilizadas para la interfaz de usuario.
  • JavaScript: Utilizado para manejar la interacción del usuario en el navegador.

Cómo Jugar

Para jugar, simplemente abre la aplicación web y sigue las instrucciones en pantalla. Haz clic en una celda del tablero para realizar tu movimiento. La IA responderá automáticamente con su movimiento. El juego termina cuando uno de los jugadores alinea tres de sus símbolos o cuando el tablero está lleno sin un ganador.
